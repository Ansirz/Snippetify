SNIPPETIFY
Description: 
  An offline app built in flutter which allows you to store "snippets" of your favorite code or just boilerplate.
Features:
- Persistence with Hive
- Functioning searching
- Copy to clipboard
- Snippet delete
What I learned:
- How to use Gridview.builder
- Implementing Animated containers
- Managing text overflows
- How to make functional search bars
Tech stack:
- Flutter
- Dart
